# react-es6-starter 

Quick away to start a React project with tests integrated

## What's inside?
  - ES6 (Babel)
  - JSX (React)
  - ESLint
  - webpack
  - Chai, Mocha & Sinon
  - Karma for tests
  - Grunt (Task runner)

## Installing Prerequisites
  - Install nodejs
  - Install npm modules (locally)
    - `npm install` (`npm i`)
  - Run
    - `grunt debug`
  - Run tests
    - `npm test`
